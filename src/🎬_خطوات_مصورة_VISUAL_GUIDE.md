## 🌟 نصائح للاستخدام

### 💡 نصيحة 1: تغيير كلمة المرور

```
1. اضغط على اسمك في الأعلى
2. اختر "الإعدادات"
3. اضغط "تغيير كلمة المرور"
4. أدخل كلمة المرور الجديدة
5. احفظ ✅
```

### 💡 نصيحة 2: إضافة مدرس

```
1. اذهب إلى "إدارة المستخدمين"
2. اضغط "إضافة مستخدم"
3. املأ البيانات:
   - الاسم: د. أحمد محمد
   - البريد: ahmad@kku.edu.sa
   - الدور: Instructor
4. اضغط "حفظ" ✅
```

### 💡 نصيحة 3: إضافة طالب

```
1. اذهب إلى "إدارة المستخدمين"
2. اضغط "إضافة مستخدم"
3. املأ البيانات:
   - الاسم: محمد عبدالله
   - البريد: mohammed@kku.edu.sa
   - الدور: Student
   - الرقم الجامعي: 441234567 ⚠️ مهم!
4. اضغط "حفظ" ✅
```
